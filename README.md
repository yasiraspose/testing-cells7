# Aspose.Cells for .NET – Agentic Code Examples

This repository contains automatically generated **Aspose.Cells for .NET** code examples.

These examples are generated by the **Aspose.Cells Product Agent** as part of the Professionalize Multi-Agent Network.

## Architecture

The generation pipeline works as follows:

1. Task Browser Agent (7001) provides categories and tasks.
2. Examples Super Agent (7050) generates code examples.
3. Aspose.Cells Product Agent:
   - builds the code
   - validates code execution
   - fixes issues automatically if needed
   - publishes working examples

Only validated examples are published.

## Repository Structure

Examples are organized by category folders.

Each category contains:
- `.cs` example files
- `agents.md` listing examples
